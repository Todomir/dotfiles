Noise Warning is the first typeface series

The name and logo of this update is completed by combining the Greek letter "ϗ", which I think is the most complicated, and the Japanese Kanji character "改" which means change or update. Both characters are pronounce as "kai".

Nosie Warning-ϗ has gain the feature of ability to change the font weight. Therefore, Nosie Warning-ϗ is a variable font!

Nosie Warning-ϗ contains 1574 characters, including Latin, Greek, Cyrillic, Hebrew, Kana and many more.(included some of the fonts have multiple choices).

Most of the characters contained in the font are not in my primary language, so please forgive me if I have offended you in any way.

It can be used for both personal and commercial purposes. But please always use the latest version of this font!

騒音警示はわれら最初のフォントシリーズです。今年の「騒音」の作字を拡張した内容です。

このアップデートの名とロゴは、一番複雑だと思うのギリシャ文字「ϗ」と漢字「改」を組み合わせで完成です。 どちらの文字も「カイ」と発音します。

騒音警示·改は、フォントの太さを変更することができるようになりました。つまり、騒音警示·改は可変フォントなのです!

騒音警示·改はラテン文字、ギリシア文字、キリール文字、ヘブライ文字、仮名といろいろを含む1574文字が含まれています（一部のフォントには複数の選択肢があります）。
